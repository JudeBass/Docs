Getting Started with UDOO

Now that you got your UDOO follow this easy steps to set everything and start making awesome stuff.

Step1: Create a bootable microSD card with UDOO OS

UDOO's Operating System and internal storage are on microSD Cards. To begin, just choose your preferred Operating System and create your microSD card.

Step2: Connect the cables and power up!

In order to use UDOO you just need to:

  2. 1- Insert the microSD card with the UDOO OS loaded
  3. 2- Connect the HDMI Cable
  4. 3- Connect Network (optional)
  5. 4- Connect Mouse and Keyboard
  6. 5- Connect Power supply and Power it Up!
UDOO will boot as soon as you connect the power supply.
UDOO comes with a 12V and 2Amp switching DC supply with a standard 5.5mm/2.1mm barrel jack. This supply is designed to work anywhere in the world, 100V-240V AC wall power but you may need a plug adapter. Further infos about Power Supplies
Now that you have a fully functional board, you can start creating awesome projects! So, what are the next steps?

Operating UDOO Remotely

UDOO can be operated also without a keyboard and a mouse attached, and also from kilometers away! You can connect UDOO to your computer in a several, easy and convenient different ways:

Connecting via SSH

SSH allows to operate UDOO remotely with a command line shell, from any computer! Learn more

Connecting via USB Serial Cable

Serial Connection allows to operate UDOO in a shell environment via USB serial cable. Learn more

Connecting via VNC Remote Desktop

Remote Desktop allows to control UDOO like you were using a keyboard, mouse and a monitor. Wherever you are. Learn more
